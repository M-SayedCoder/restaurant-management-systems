# Burgerizza - Restaurant Management System

Burgerizza is a web-based restaurant management and online ordering application designed for a single restaurant. It provides customers with a simple and fast way to browse menus, place orders, and reserve tables, while giving managers and admins the tools to efficiently manage daily operations.

---

## Project Overview

- Customer: Browse menu, place online orders, reserve tables, submit reviews.
- Manager: Manage menu items, track orders, handle reservations, monitor performance.
- Admin: Approve critical updates, view reports and statistics.

---

## Pages

### Authentication Pages
- Login Page: Email and password input, role-based redirection.
- Register Page: Name, email, password, role selection.

### Customer Pages
- Menu Page: Display menu items with image, name, price, and add to cart button.
- Cart / Order Page: Order summary, quantity adjustment, remove items, checkout.
- Reservation Page: Select date, time, and number of guests, confirm booking.

### Manager Pages
- Manager Dashboard: Overview of orders, reservations, pending actions.
- Menu Management: Add, edit, or delete menu items with image upload.
- Orders Management: View orders and update status (Preparing, Completed).
- Reservations Management: View bookings, accept or reject reservations.

### Admin Pages
- Admin Dashboard: Orders overview, reservation statistics, performance reports.

## Design System

- Primary Color: Orange #FF6F00 (Buttons, highlights)
- Secondary Color: Navy #0D1B2A (Navbar, headers)
- Background: White #FFFFFF
- Font: Poppins or Roboto
- Style: Rounded cards, soft shadows, minimal and responsive design
